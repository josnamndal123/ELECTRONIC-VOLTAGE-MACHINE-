## Title

To measure DC voltage using Arduino and display the measured voltage value accurately.

## Components Required

- Arduino Uno
- Voltage Sensor Module (0–25V)
 - Breadboard
- Jumper Wires
- LCD Display (Optional)
- DC Battery/Power Supply

## Working Principle

The voltage sensor measures the input voltage and sends the signal to the Arduino. Arduino converts the analog value into voltage and displays the result on the Serial Monitor or LCD screen.
