## Project title

To measure DC voltage using Arduino and display the measured voltage value accurately.

## Components Required

- Arduino Uno
- Voltage Sensor Module (0–25V)
 - Breadboard
- Jumper Wires
- LCD Display (Optional)
- DC Battery/Power Supply

## Working Principle

The voltage sensor measures the input voltage and sends the signal to the Arduino. Arduino converts the analog value into voltage and displays the result on the Serial Monitor or LCD screen.

## Applications

- Battery Testing
- Power Supply Monitoring
- Electronics Laboratories
- Industrial Voltage Measurement
  
## Advantages

- Low Cost
- Easy to Build
- Accurate Measurement
- Real-Time Display
  
## Conclusion

The Electronic Voltage Measurement Machine is a simple Arduino-based project used to measure DC voltage. It provides accurate voltage readings and is useful for students, laboratories, and electronic testing applications.
